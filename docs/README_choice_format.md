# Seat Arrangement Problem — Input File Format

## 概要
此輸入檔描述一個 **座位安排問題**：

- 一個座位區為 `R × C` 的矩陣。  
- 總共有 `R × C` 名學生，每位學生對應一個唯一座位。  
- 每位學生對「可接受的座位區域」、「想要的鄰座」以及「不想要的鄰座」都有偏好。  

## 格式定義

### 第一行
```
R C
```
- `R`：座位的列數 (rows)  
- `C`：座位的行數 (columns)  
- 總學生數 = `R × C`  

### 每位學生的描述 (四行一組)
1. **學生編號**  
   - 一個整數，從 1 開始，依序到 `R × C`。  

2. **Preferred zone (可接受的座位區域)**  
   - 由一個或多個長方形區塊描述，每個區塊格式為：  
     ```
     r1 c1 r2 c2
     ```  
     表示左上角 `(r1, c1)` 到右下角 `(r2, c2)` 的矩形（包含邊界）。  
   - 多個區塊以 **分號 (;)** 隔開。  
   - 範例：  
     ```
     1 1 2 2 ; 3 2 3 3
     ```  
     表示可接受 `(1,1)` 到 `(2,2)` 的 4 格，以及 `(3,2)` 到 `(3,3)` 的 2 格。  

3. **想跟誰坐 (Want to sit with)**  
   - 學生編號列表，以空白分隔。  
   - 若沒有，填 `-`。  
   - 範例：  
     ```
     2 3 4
     ```  

4. **不想跟誰坐 (Don’t want to sit with)**  
   - 學生編號列表，以空白分隔。  
   - 若沒有，填 `-`。  
   - 範例：  
     ```
     5 6
     ```

## 注意事項
- 所有座標 `(r, c)` 需滿足：  
  ```
  1 ≤ r ≤ R
  1 ≤ c ≤ C
  ```
- 每位學生恰好描述一次，順序為 1 到 `R × C`。  
- 「想跟誰坐」與「不想跟誰坐」的學生編號必須在合法範圍內 (1 … R×C)。
